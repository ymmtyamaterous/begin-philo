# UI / 画面設計書

## 1. デザインシステム

### 1.1 デザインコンセプト
`docs/user/sample.html` のデザインを踏襲する。
- **コンセプト**: 古書・和紙を想起させる落ち着いたトーン。知的・上品な雰囲気。
- **雰囲気**: アンティーク / リテラリー / アカデミック
- **フォント**: Noto Serif JP（本文）、Shippori Mincho（見出し）、Cormorant Garamond（装飾・数字・英語引用）、DM Serif Display（英字タイトル装飾）

### 1.2 カラーパレット

| 変数名 | 値 | 用途 |
|-------|---|------|
| `--ink` | `#1a1209` | メインテキスト、背景（ダーク） |
| `--paper` | `#f5f0e8` | メイン背景（和紙） |
| `--aged` | `#e8dfc8` | セカンダリ背景（経年した紙） |
| `--accent` | `#8b4513` | アクセント（煉瓦・朱） |
| `--gold` | `#c9a84c` | ゴールドアクセント |
| `--muted` | `#7a6a52` | サブテキスト |
| `--light-muted` | `#b5a68a` | 薄いサブテキスト |
| `--red` | `#8b2020` | 赤アクセント（ホバー等） |
| `--green-dark` | `#2d4a2d` | 無料バッジ等 |

### 1.3 タイポグラフィ

| 要素 | フォント | サイズ | 備考 |
|------|---------|--------|------|
| H1（ヒーロータイトル） | Shippori Mincho | clamp(3rem, 5vw, 5rem) | 重量800 |
| H2（セクション見出し） | Shippori Mincho | clamp(2rem, 3.5vw, 2.8rem) | 重量600 |
| H3（カードタイトル） | Shippori Mincho | 1.1〜1.3rem | 重量600 |
| 本文 | Noto Serif JP | 0.85〜1rem | 重量300〜400 |
| ラベル・キャプション | Noto Serif JP | 0.72〜0.78rem | 大文字・letter-spacing: 0.15〜0.35em |
| 装飾数字・英語引用 | Cormorant Garamond | 1〜6rem | Italic |

### 1.4 アニメーション
- スクロール連動の `reveal` アニメーション（`IntersectionObserver`）
  - `opacity: 0 → 1`、`translateY(24px) → 0`、duration: 0.7s
- グリッド内カードは stagger（子要素ごとに 0.07s 遅延）
- ホバー時: `translateY(-2px〜-4px)` + `box-shadow`

### 1.5 レスポンシブブレークポイント
| ブレークポイント | 範囲 | 主な変化 |
|----------------|------|---------|
| Mobile | 〜 640px | 1カラム、ヒーロー右側非表示 |
| Tablet | 641px 〜 900px | 2カラムグリッド |
| Desktop | 901px 〜 | フルレイアウト |

## 2. 共通コンポーネント

### 2.1 Header
- 固定ヘッダー（`position: fixed`）、`backdrop-filter: blur(8px)`
- **左**: ロゴ `哲学の庭`（庭の文字をアクセントカラーで）
- **中央**: ナビゲーション（学習コース / 哲学者 / テーマ / 読む）
- **右**: 無料バッジ ＋ ログインボタン / ユーザーメニュー
- ナビリンクにホバー時の下線アニメーション

### 2.2 Footer
- 背景: `--ink`（ダーク）
- **グリッド**: ブランドエリア（2fr）、コンテンツ（1fr）、東洋哲学（1fr）、サイトについて（1fr）
- リンクホバー: `--gold` カラーに変化

### 2.3 SectionHeader
- セクションラベル（小文字大文字・letter-spacing大）
- セクションタイトル（Shippori Mincho）
- 区切り線（60px、アクセントカラー）
- 説明文

### 2.4 RevealWrapper
- `IntersectionObserver` を使ったスクロールアニメーションラッパー

### 2.5 SearchBar
- 入力フィールド + 検索ボタン
- 境界線: `1px solid rgba(139,69,19,0.25)`
- 背景: `rgba(255,255,255,0.6)`

## 4. インタラクション仕様

### 検索
- ヘッダー内の検索バー（モバイルでは検索アイコンタップで展開）
- 入力から300ms後にデバウンスで `/api/rpc` の `search.query` を呼び出し
- 結果はドロップダウン形式でリアルタイム表示（最大5件/カテゴリ）
- Enter キーまたは全件表示ボタンで `/search` ページへ遷移

### ブックマーク
- 各コンテンツページにブックマークボタン（ハートアイコン）
- 非ログイン時: クリックでログインページへリダイレクト
- ログイン時: 楽観的UI更新（即座にUIを更新し、バックグラウンドでAPI呼び出し）

### 学習進捗
- レッスン完了ボタンクリック → API呼び出し → プログレスバー更新
- コース完了時: トースト通知「コース「XXX」を完了しました！」

### 名言ローテーション
- ヒーローセクションの円形引用ボックス: 4秒ごとにフェードして名言切り替え

## 5. Tailwind CSS / shadcn/ui 対応方針

- shadcn/ui コンポーネントを積極活用: `Button`, `Card`, `Input`, `Badge`, `Tabs`, `Progress`, `Skeleton`
- カスタムカラーは `tailwind.config.ts` の `extend.colors` に CSS 変数として定義
- 和紙テクスチャのノイズオーバーレイは `body::before` で実装

```css
/* tailwind.config.ts で定義するカスタムカラー */
colors: {
  ink: 'var(--ink)',
  paper: 'var(--paper)',
  aged: 'var(--aged)',
  accent: 'var(--accent)',
  gold: 'var(--gold)',
  muted: 'var(--muted)',
}
```
